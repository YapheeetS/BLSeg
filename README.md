## LLSeg 

PyTorch implementation of Semantic Segmentation Model

---

#### Requirement

* python 3.x
* pytorch >= 1.0.0

---

#### Support Model

* [FCN]
* [U-Net]
* [DeepLab v3+]

---

#### Support Backbone

* [VGG16]
* [MobilenNet v1]
* [ResNet50] (Modified according to [Bag of Tricks])
* Modified Aligned Xception for [DeepLab v3+]
